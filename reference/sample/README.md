# CooMate - 你的智能思考引导者

这是一个基于极简主义设计、并吸纳了 Claude 官方经典配色的前端界面模板，主要目标是展现 "CooMate 交互式 AI 平台" 的基本框架及四项核心功能（深度研究、创意发散、情感分析、决策分析）。

## 目录结构
\`\`\`
temp1/
├── index.html          # 入口HTML，构建应用骨架与四板块
├── css/
│   └── style.css       # 统一样式文件，支持多样式主题及CSS变量
├── js/
│   └── main.js         # 前端交互逻辑（侧边栏、发送消息模拟、主题切换等）
├── assets/             # 图片、字体等静态资源
│   └── images/
└── README.md           # 项目说明文件
\`\`\`

## 核心特性
- **UI 主题配置**：提供蓝白配色（默认）、Claude风格（亚麻米白底色），以及暗夜模式，只需修改即可一键切换。主题配置依靠 CSS `:root` 以及 `.theme-*` 响应。
- **四个思考板块（主入口）**：以大块极简卡片形式，直观切入用户诉求。包含`深度研究`、`创意发散`、`情感分析`、`决策分析`。
- **智能问询模式（Plan 模式）**：在 AI 提问时暂时挂起聊天流，提供预设的回答建议以及“其他”选项，进一步引导与降低用户交互门槛。
- **全局双侧边栏布局**：左侧为历史与用户登录（底层），右侧为可折叠的 **思考进程面板**（右上角图谱按钮控制），展示当前分析进度。
- **流畅的交互感知**：对话流随内容增长而滚动，发信栏能够根据内容的高度自适应拓展。

## 使用方法
由于仅包含纯静态文件（HTML / CSS / JS），无需搭建编译环境。直接使用浏览器打开 `index.html` 即可预览完整页面。
